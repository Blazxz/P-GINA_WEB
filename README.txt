Descripción del proyecto y funcionales: Esto es una página web que usa los lenguajes de etiquetas HTML y CSS cuya temática es la saga de Crash Bandicoot, la página web usa desde listas, animaciones, tablas, una imagen de fondo, enlaces que llevan a las diferentes secciones como productos, contacto y la wiki de los juegos en las tarjetas, una página de entrada como es index.html. Distintas animaciones con keyframes en fragmento destacado

Cómo van a usarlo los diferentes usuarios: Pueden usar la página embebida como una muestra de la página productos, usar los diferentes enlaces para navegar en las diferentes páginas, acceder a la wiki de los juegos, una imagen de crash bandicoot y un formulario para registrarse.

Autores del proyecto:Daniel Blazquez Marín, Samuel Garcerán López.
